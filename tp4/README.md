Levantar los contenedores con:

```bash
./run-compose.sh
```

O bien con

```bash
docker-compose -p <NOMBRE_PROYECTO> up -d
```

El nombre del proyecto utilizado es **universidad**

---

Para eliminarlos:

```bash
./stop-compose.sh
```

O bien con

```bash
docker-compose -p <NOMBRE_PROYECTO> down
```
